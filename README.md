# Hotel Management System

A desktop-based Hotel Management System developed using **Java** and **JavaFX**. The application provides separate interfaces for administrators and receptionists to manage hotel operations, including room management, reservations, guest check-in/check-out, billing, and occupancy tracking.

---

## Features

- User authentication with role-based access (Admin & Receptionist)
- Room management
- Room booking and reservation
- Guest check-in and check-out
- Billing and invoice generation
- Dashboard with room availability and occupancy statistics
- Dark and Light mode support
- File-based persistent data storage

---

## Tech Stack

- Java
- JavaFX
- Maven
- CSS (JavaFX Styling)
- File-based Storage
